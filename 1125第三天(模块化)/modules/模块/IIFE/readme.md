##下面是关于IIFE部分的理解：
> * :
```
首先这种格式就是一个立即执行的函数方式：
（function（）{###}）（）    //用一个()扩住一个匿名函数，将其转化为函数表达式；第二个()表示将该函数表达式进行执行。
```

> * :
```
作用是什么？ 
  通常我们都会把代码写在$(function(){  ### })中。在函数自执行中通常也有这种写法的效果，能立即执行该函数。
  1.创建块级（私有）作用域，避免了向全局作用域中添加变量和函数，因此也避免了多人开发中全局变量和函数的命名冲突
  2.IIFE中定义的任何变量和函数，都会在执行结束时被销毁。这种做法可以减少闭包占用的内存问题，因为没有指向匿名函数的引用。只要函数执行完毕，就可以立即销毁其作用域链了。

```

> * :
```
下面可已看看一个简单的模块化例子，基础知识是函数自执行的方式向外面暴露window变量：
(function(window){
    const mi = 'xiaomi'
    function foo(){
      alert(111);
    }
    window.foo = {
      mi,foo
    }
})(window)

但是有一点需要注意的是:如果该程序需要依赖于jquery进行操作，那么在引入该js前必须先引入jquery插件，而且顺序不能错。
```
